# AI Assistance Policy

Causlane allows AI coding assistants for design notes, scaffolding, refactoring,
documentation, test generation and maintenance work.

AI tools are not authors, maintainers, reviewers or signers of this project.
Human contributors remain responsible for all accepted changes, including
correctness, security, licensing and provenance.

## Commit Attribution

Use a human or organization-controlled identity as Git author and committer.

Do not add AI tools as `Co-authored-by:` trailers. Material AI assistance may be
disclosed with an `Assisted-by:` trailer or in the pull request body:

```text
Assisted-by: OpenAI ChatGPT
```

If the project uses sign-off trailers, only humans may add `Signed-off-by:`.

## Contributor Responsibility

Contributors using AI assistance must:

- review and understand submitted code;
- verify that generated text and code are suitable for the project license;
- avoid copying hidden, proprietary or otherwise unavailable source material;
- run the required checks for the change;
- disclose material AI assistance when appropriate;
- ensure no prompts, credentials or private context packs are committed.
