## Evidence

`examples/release-orchestration` is a standalone in-repo reference integration
slice for CI/CD release orchestration. It models CI gates, package-list review,
publish dry-run planning and downstream smoke planning as a dependency graph
through host-dispatch v2 and the linear runtime worker.

The example is intentionally bounded: it does not publish crates, sign tags,
read tokens, mutate crates.io, create GitHub releases or operate as a durable CI
worker. `docs/product-track/reference-integration-matrix.json` records those
deferred claims and is checked by `tools/reference-integration-check`.
