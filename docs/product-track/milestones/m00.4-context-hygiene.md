# M00.4 — Context hygiene

- **Stage:** S00 — Контекст, назначение и продуктовая рамка
- **Status:** `ongoing`
- **Purpose:** Контекст-паки, generated files, formal artifacts и секреты должны иметь hygiene/check tools.

## Required outputs

- normative doc/spec update;
- machine-readable artifact where applicable;
- tests/replay/formal receipt where applicable;
- coverage/readiness status update where applicable;
- risk/limitation note if not fully enforced;

## Acceptance checklist

- [ ] Implementation or generated artifact exists.
- [ ] Docs and ADRs match the implementation.
- [ ] No protocol-critical behavior is prose-only.
- [ ] Relevant negative controls exist.
- [ ] Stale/coverage/release gates updated.
