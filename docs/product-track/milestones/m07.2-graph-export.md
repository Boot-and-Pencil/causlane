# M07.2 — Graph export

- **Stage:** S07 — Observability, explainability и DX
- **Status:** `done_or_near_done`
- **Purpose:** Mermaid/DOT/JSON graph slices with blockers/witnesses/leases.

## Delivery note

M07.2 lands a CLI graph exporter over the same typed graph snapshot consumed by
M07.1:

- `causlane graph export --graph <graph.yaml|json> --format json [--op
  <action_id>:<op_index>] [--out <path>]` emits a schema-versioned graph slice
  with lanes, ops, statuses, blockers, witness labels, lease labels and edges.
- `--format mermaid` and `--format dot` render the same export model for visual
  inspection.
- `--op` narrows the export to the target op plus direct blocker evidence; without
  `--op`, the full snapshot is exported.

The exporter delegates readiness and blockers to `GraphIndex`, `select_frontier`
and `why_not_parallel_from_index`; it does not duplicate scheduler logic.
Witnesses and leases are opaque snapshot labels for display in this milestone;
validation remains in the existing replay/barrier/capability paths.
