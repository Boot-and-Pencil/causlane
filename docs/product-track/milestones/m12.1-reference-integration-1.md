# M12.1 — Reference integration 1

- **Stage:** S12 — Beta integrations и ecosystem validation

- **Purpose:** Rust service with API+worker+audit+projection.

## Evidence

`examples/reference-integration` is a standalone in-repo reference integration
slice. It submits host tasks through host-dispatch v2, drains them with the
linear runtime worker, appends audit events through the runtime audit adapter and
authorizes/redacts a projection read through the guarded projection helper.

This is a hardening slice, not a production service claim: durable queues,
distributed lease coordination, HTTP/database integration and retry policy remain
outside this executable example.
