# M07.3 — Tracing connector

- **Stage:** S07 — Observability, explainability и DX
- **Status:** `done_or_near_done`
- **Purpose:** Structured action/op spans; logs derived, not truth.

## Delivery note

M07.3 adds a typed, dependency-free `AuditEvent -> TraceSpan` projection in
`causlane-core` and a runtime audit-log wrapper that emits one derived span only
after the authoritative audit append succeeds. Sink failures are telemetry
failures and do not block the audit append.

The mapping has a single implementation in core. Runtime adapters and future
exporters must consume that projection rather than reclassifying audit events.

## Limitation

This milestone does not add an OpenTelemetry/OTLP exporter or a dependency on
the Rust `tracing` ecosystem. External exporter wiring remains M07.4.
